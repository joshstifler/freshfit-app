
# Instruções de Deploy (Streamlit Cloud)

1. Crie uma conta em https://streamlit.io/cloud
2. Suba este projeto no GitHub
3. No repositório, crie uma pasta `.streamlit` e dentro dela adicione `secrets.toml` com:
[discord]
webhook_url = "SUA_WEBHOOK_AQUI"
4. No Streamlit Cloud, configure este repositório para deploy
5. O app ficará online e pode ser acessado de qualquer celular
